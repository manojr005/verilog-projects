# Verilog Digital Design Modules

##  Overview

This repository contains Verilog HDL implementations of fundamental digital design modules, including combinational, sequential, and memory blocks.
Each module is provided along with a corresponding testbench for functional verification.

---

##  Modules Included

### Combinational Circuits

* Logic gates
* Multiplexer (MUX)
* Demultiplexer (DEMUX)
* Encoder / Decoder
* Comparator
* Half Adder / Full Adder

---

### Sequential Circuits

* SR Flip-Flop
* JK Flip-Flop
* D Flip-Flop
* T Flip-Flop
* Counters


---

### Memory Blocks

* RAM
* FIFO

---

## 🧪 Testbenches

Testbenches are included for each module to verify functionality through simulation.
